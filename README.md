## 📖 О проекте

**RUST: Survival Edition** — одиночная игра, вдохновлённая механиками *Rust*.  
Исследуйте опасные локации, собирайте ресурсы, изучайте чертежи, создавайте оружие и стройте собственную базу.  
Динамическая среда, смена дня и ночи, голод, радиация и рейдеры не дадут расслабиться.

Проект разработан с использованием **стандартной библиотеки Python** — никаких внешних зависимостей.  
Весь прогресс автоматически сохраняется в локальную базу данных SQLite.

## ✨ Ключевые возможности

| Категория | Описание |
|-----------|----------|
| 🖥️ **Графический интерфейс** | Удобное окно на Tkinter с живой консолью, кнопками действий и всплывающими панелями. |
| 📜 **Система чертежей** | Изучайте рецепты за собранный скрап, открывая доступ к новому оружию и постройкам. |
| 🔧 **Крафт** | Создавайте инструменты, броню, оружие, медицинские предметы и строительные блоки. |
| 🏗️ **Строительство базы** | Возводите стены, двери, верстаки на сетке 5×5 — база влияет на защиту от рейдов. |
| 🗺️ **Открытый мир** | 4 локации: Пляж, Лес, Горы и Радиоактивный город. Каждая со своими рисками и ресурсами. |
| ⚔️ **Боевая система** | Охотьтесь на животных (волк, кабан, медведь) и отбивайтесь от рейдеров в пошаговых боях. |
| ☢️ **Факторы выживания** | Следите за голодом, жаждой и радиацией — их истощение приводит к потере здоровья. |
| 🌙 **Цикл дня и ночи** | Ночью шанс встретить врага увеличивается. Время суток влияет на геймплей. |
| 🛡️ **Рейды на базу** | Периодически вашу базу пытаются ограбить. Прочность построек определяет исход. |
| 💾 **Надёжное сохранение** | База данных SQLite хранит состояние игрока, инвентарь, чертежи и постройки. |

## 🚀 Быстрый старт

### Требования

- **Python 3.7 или новее** (протестировано на 3.14)
- Все необходимые модули (`tkinter`, `sqlite3`, `json`, `random`, `time`) входят в стандартную поставку Python.

> **Примечание для Linux:** если Tkinter отсутствует, установите его через пакетный менеджер:
> ```bash
> sudo apt install python3-tk   # Debian/Ubuntu
> sudo dnf install python3-tkinter  # Fedora
> ```
> Пользователям macOS и Windows дополнительных действий не требуется.
