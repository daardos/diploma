## ✨ Особенности

- **🕹️ GUI на Tkinter** — кнопки, списки, консоль, всплывающие окна.
- **📜 Изучение чертежей** за собранный скрап.
- **🔧 Крафт** оружия, инструментов, брони, строительных блоков.
- **🏗️ Строительство базы** с отображением на 2D‑карте (вид сверху).
- **⚔️ Охота** на животных (волк, кабан, медведь) и бой с рейдерами.
- **🌍 Открытый мир** — 4 локации (пляж, лес, горы, радиоактивный город).
- **☢️ Радиация, голод и жажда** — следи за состоянием персонажа.
- **🌙 Смена дня и ночи** — влияет на появление врагов.
- **🛡️ Рейды на базу** — защищай свою постройку от грабителей.
- **💾 Сохранение в SQLite** — весь прогресс хранится локально.

## 🚀 Установка и запуск

### Требования
- **Python 3.7 или выше** (разработано на 3.14)
- Стандартные библиотеки: `tkinter`, `sqlite3`, `json`, `random`, `time` (входят в любую установку Python)

### Запуск
1. Клонируй репозиторий или скачай ZIP и распакуй.
2. Убедись, что в папке с игрой лежат все файлы (см. [Структуру проекта](#-структура-проекта)).
3. Запусти главный скрипт:
   ```bash
   python main.py
   ```
4. Если окно не появилось – убедись, что Tkinter установлен (обычно входит в Python).

### Первый запуск
При первом запуске автоматически создаётся база данных `rust_game.db` и новый игрок по имени **Facepuncher** с начальными параметрами.  
Все изменения сохраняются сразу.

## 📁 Структура проекта

```
rust_game/
├── main.py                # Точка входа
├── database.py            # Работа с SQLite (CRUD для игрока, инвентаря, построек)
├── game_logic.py          # Игровые механики (день/ночь, голод, крафт, бой, рейды)
├── ui.py                  # Графический интерфейс (окна, консоль, кнопки)
├── buildings.py           # Окно 2D‑вида базы
├── game_data.json         # Все предметы, чертежи и рецепты крафта
└── README.md              # Этот файл
```

Все модули связаны между собой, глобальный объект игрока импортируется из `database.py`.

## 🎮 Как играть

### Интерфейс
- **Верхняя строка** — HUD: имя, здоровье, голод, жажда, скрап, время суток, локация.
- **Нижняя консоль** — сюда выводятся все результаты действий, урон, сообщения о событиях.
- **Центральная панель** — кнопки действий.

### Основные действия
| Кнопка | Описание |
|--------|----------|
| **🔨 Искать компоненты (Лутать)** | Найти скрап в текущей локации. На РТ (рад. город) есть шанс получить урон радиацией. |
| **🪓 Собрать ресурсы** | Добыть дерево, камень, руду (зависит от локации). Инструменты дают бонус. |
| **🎒 Инвентарь** | Показывает все предметы и их количество. |
| **🔧 Крафт** | Создать предмет, если изучен чертёж, хватает ингредиентов и есть нужный верстак в инвентаре. |
| **📜 Чертежи** | Изучить новый чертёж за скрап (если не изучен). Некоторые даны по умолчанию. |
| **🏗️ Строительство** | Построить элемент базы (стена, дверь, верстак и т.д.) из инвентаря. Постройка автоматически размещается на 2D-сетке. |
| **🗺️ Карта** | Переместиться в другую локацию. При перемещении может произойти случайное событие (нападение зверя, рейдер, радиационный шторм). |
| **⚔️ Охота** | Выйти на поиски случайного зверя. Бой в отдельном окне. |
| **🏰 Моя база** | Открывает вид базы сверху (сетка 5×5). |
| **🛡️ Проверить рейд** | Шанс, что на вашу базу нападут рейдеры. Если нет стен или дверей – могут украсть ресурсы. |

### Бой
При встрече с врагом (животное, рейдер) появляется панель с кнопками:
- **⚔️ Атаковать** — нанести урон (зависит от оружия в инвентаре).
- **💉 Лечиться** — использовать медицинский шприц / большую аптечку.
- **🏃 Убежать** — шанс 50% сбежать, иначе враг атакует в ответ.

После победы выпадает добыча (мясо, кожа, кости, скрап).

### Выживание
- **Голод** и **жажда** уменьшаются со временем (каждые 3 секунды). Если показатель достигает 0, вы начинаете терять здоровье.
- **День** сменяется **ночью** каждые 30 секунд. Ночью выше шанс встретить врага.
- При **смерти** вы возрождаетесь:
  - Если построена **кровать** — на базе, теряете 20% скрапа.
  - Без кровати — на **Пляже**, теряете 50% скрапа.

### Строительство и база
- Постройки занимают клетки на поле 5×5.
- Чем больше стен и дверей – тем выше защита от рейдов.
- Верстаки нужны для крафта продвинутых предметов.
- Просмотреть базу можно через **🏰 Моя база**.

## 💾 База данных

Игра использует SQLite (`rust_game.db`) с таблицами:
- `players` – состояние игрока (hp, scrap, голод, локация, время суток).
- `inventory` – инвентарь (предмет, количество).
- `blueprints` – изученные чертежи.
- `buildings` – построенные объекты с координатами.

При повреждении базы можно просто удалить файл `rust_game.db` – при следующем запуске создастся новый, с чистым прогрессом.
